# AS-25: Quick Action button framework + modal system

## Scope
Build a reusable Quick Action button + modal system that sits above the chat input in every agent mode.

## Components

### `apps/web/src/components/QuickActionBar.tsx`
```typescript
type QuickAction = {
  id: string;
  label: string;
  icon?: string;
  fields: QuickActionField[];
  promptTemplate: string; // template with {{field_id}} placeholders
};

type QuickActionField = {
  id: string;
  label: string;
  placeholder: string;
  type: "text" | "textarea" | "select";
  required: boolean;
  options?: string[];
};

type Props = {
  agentId: string;
  agentLabel: string; // e.g. "Research Assistant" â€” shown as green label
  actions: QuickAction[];
  onSubmit: (prompt: string) => void; // injects into chat
};
```

### `apps/web/src/components/QuickActionModal.tsx`
- Modal overlay (dark, centered, max-w-lg)
- Title: action label
- Form fields from `QuickAction.fields`
- "Start" / "Cancel" buttons
- On submit: replace `{{field_id}}` placeholders in `promptTemplate` with values, call `onSubmit(prompt)`
