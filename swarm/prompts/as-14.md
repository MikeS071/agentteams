# AS-14: Sidebar redesign — narrow tile icons with labels

## Scope
Convert the workspace sidebar from a wide text-based nav into a narrow icon-tile sidebar.

## Current state
File: `apps/web/src/app/dashboard/layout.tsx`
- Sidebar is `w-56` (224px) with text labels
- Nav items: Chat, Channels, Hands (→ AI Agents), Agents, Approvals, Swarm, Usage, Billing
- Settings items: Profile, General, Deploy

## New design
- Sidebar width: `w-16` (64px) — narrow icon strip
- Each nav item becomes a square tile (48x48 or 44x44):
  - Small icon (Lucide icon or emoji) centered
  - Tiny label below (text-[10px], truncated)
  - Active state: highlighted background + colored border-left or ring
- Logo at top: just the icon/emoji, no text
- User menu at bottom: avatar circle only, click to expand

## Icon mapping (use Lucide React icons):
- Chat → `MessageSquare`
- Channels → `Radio`
- AI Agents → `Bot`
- Approvals → `CheckCircle`
- Swarm → `Network`
- Usage → `BarChart3`
- Billing → `CreditCard`
- Profile → `User`
- Settings → `Settings`
- Deploy → `Rocket`

## Implementation
1. Install lucide-react if not already: check package.json first
2. Reduce sidebar width from `w-56` to `w-16`
3. Replace text links with icon tiles
4. Add tooltip on hover showing full label (use `title` attribute or a tooltip component)
5. Keep the mobile sidebar behavior (hidden on mobile, hamburger to open)

## Styling
- Tiles: `flex flex-col items-center justify-center gap-0.5 rounded-lg p-2 hover:bg-[#131320]`
- Active: `bg-[#131320] border-l-2 border-[#a29bfe]` or ring
- Label: `text-[10px] text-gray-500 leading-tight`
- Icon: `h-5 w-5 text-gray-400` (active: `text-[#a29bfe]`)
