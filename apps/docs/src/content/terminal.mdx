# Terminal

AgentSquads includes a secure web terminal for supervised operations.

## Access

1. Go to `Dashboard -> Terminal`.
2. Start a session scoped to your workspace.
3. Run approved commands within configured policy limits.

## Available Commands

Common command categories:

- Runtime checks: process health and logs
- Workspace operations: workflow execution tools
- Deployment helpers: build and release checks
- Diagnostics: usage, latency, and failure inspection

## Safety

- Sessions are audited and time-bound.
- Sensitive variables are masked in logs.
- High-risk commands require elevated policy approval.
