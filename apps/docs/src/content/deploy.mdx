# Deploy

Deploy user apps and integrations through Vercel and Supabase connections.

## Connect Vercel

1. Open `Dashboard -> Settings -> Deploy -> Vercel`.
2. Authorize your Vercel account.
3. Select project and target environment.
4. Confirm callback and connection status.

## Connect Supabase

1. Open `Dashboard -> Settings -> Deploy -> Supabase`.
2. Authorize and choose project.
3. Map environment variables and secrets.
4. Validate database and edge function access.

## Deploy Flow

1. Build deploy bundle.
2. Validate required keys and resources.
3. Trigger deploy and monitor logs.
4. Verify post-deploy health checks.
