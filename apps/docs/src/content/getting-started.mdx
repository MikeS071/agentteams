# Getting Started

This guide walks through a first-time setup: signup, first chat, Telegram connection, and your first workflow run.

## 1. Sign Up

1. Open AgentSquads and create an account.
2. Verify your email and complete workspace onboarding.
3. Choose a workspace name and default region.

## 2. Start Your First Chat

1. Go to `Dashboard -> Chat`.
2. Select a starter agent.
3. Send a test prompt such as:

```text
Summarize the top 3 onboarding steps for my customer.
```

## 3. Connect Telegram

1. Open `Dashboard -> Settings -> Channels -> Telegram`.
2. Create a bot with `@BotFather` and copy the API token.
3. Paste token in AgentSquads and validate webhook.
4. Send a message to your bot and confirm it appears in conversation history.

## 4. Run Your First Workflow

1. Open `Dashboard -> Workflows`.
2. Choose a guided template such as lead qualification or support triage.
3. Add trigger inputs and run a test execution.
4. Review output and rerun with updated parameters.
